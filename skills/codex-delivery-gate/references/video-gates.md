# Video Gates

## Check

- Duration, aspect ratio, frame rate, and resolution match target platform.
- Audio exists when expected and is synchronized.
- Key frames match storyboard.
- Captions/text are readable and not clipped.
- Export file opens and plays.

## Tools

- Prefer deterministic render commands such as Remotion/ffmpeg.
- Use Computer Use for desktop editing/export tools when no programmatic path exists.
- Use imagegen only for generated raster assets, thumbnails, backgrounds, or storyboard frames.

## Evidence

- Render command or app used.
- Output path.
- Duration/resolution.
- Key-frame or playback QA result.
